# File Handling and Dictionary Operations

## Overview
This project documents Python work focused on reading and writing files, creating folders, and using dictionaries for structured data storage and lookup. The assignment includes reading a text file into a list, writing lines into a file, creating a dictionary, iterating through key-value pairs, and using dictionary lookup with fallback values.

## Tools Used
- Python

## What I Did
- Read a text file into a list while removing newline characters
- Created a folder and wrote multiple lines into a text file
- Built a dictionary using key-value pairs
- Iterated through dictionary items and printed readable output
- Used `.get()` to return values or a default fallback
