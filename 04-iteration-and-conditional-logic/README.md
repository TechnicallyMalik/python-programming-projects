# Iteration and Conditional Logic

## Overview
This project documents Python work focused on loops, iteration, filtering, and conditional logic. The assignment includes printing numbers with a while loop, identifying values divisible by 3, finding the smallest number in a list without using `min()`, and counting digits and letters in a string.
