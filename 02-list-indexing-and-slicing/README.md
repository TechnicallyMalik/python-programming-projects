# List Indexing and Slicing

## Overview
This project documents introductory Python work focused on accessing and slicing list elements. The assignment includes retrieving the first element of a list, extracting the first three elements, returning the last three elements, checking whether two lists share a common element, comparing exam scores to determine progress, and evaluating product defects using conditional logic.

## Tools Used
- Python

## What I Did
- Accessed individual list elements using indexing
- Extracted subsets of a list using slicing
- Checked whether two lists had at least one common element
- Compared two exam scores using a conditional statement
- Built a simple program to identify defective products based on type and weight

- Basic set operations
